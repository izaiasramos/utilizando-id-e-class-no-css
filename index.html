2 FORMAS DE SELECIONAR ELEMENTOS ESPECIFICOS DO HTML PARA ALTERAR NO CSS

-USANDO O ID
O problema do método css externo,é que se no seu html tiver mais de 1 elementos h4 por exemplo,e no css você atribuir o css para o h4,todos os h4 serão alterados pelo css ao mesmo tempo,então se voce quiser alterar somente um h4 especificamente você tem duas formas de fazer isso.

Uma das formas é utilizando um id no h4 por exemplo,e dar um nome específico para essa id desse h4 que você quer alterar.

ficaria assim no html: < h4 id="da um nome para essa id" > cabecalho < / h4 >

ficaria assim no css: #da um nome para essa id { color:blue; }
no css você chama a id com um #depois o nome da id,assim você altera somente 1 h4 por vez,ou seleciona apenas oque quiser.

-USANDO O CLASS

A forma mais utilizada para fazer isso é usando o class,ela pode ser usada quantas vezes você quiser,mesmo com o mesmo nome.

ficaria assim no html: < h1 class="da um nome para essa class"> cabecalho < /h1 >

ficaria assim no css: .da um nome para essa class { color:blue; }

No css você chama a class com um . (um ponto) seguido do nome que você deu para a class.

Se eu quiser tambem posso selecionar o id e o class para altera-los juntos ao mesmo tempo.
ficaria assim no css: #nomedaid, .nomedaclass { color:blue; fonte-size:25px;} chamo a id e a class ao mesmo tempo no css para alterar-los juntos somente separando eles por uma , (uma virgula).
